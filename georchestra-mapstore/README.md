# About

A simple & minimalist docker composition which aims to be able to boot
a [geOrchestra Mapstore2](https://github.com/georchestra/mapstore2-georchestra/) in a geOrchestra's context.

# Usage

Once the composition is launched, simply open [http://localhost:8080/mapstore/](http://localhost:8080/mapstore/) in your browser.

You can use testadmin/testadmin to log in as an administrator, testuser/testuser to log in as a regular user.
