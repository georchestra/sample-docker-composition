# Intent

Showcasing the use of Geonetwork-UI's datahub along with the Hugo static site generator,
in order to provide a cheap content management system.

# How to use

1. launch the docker composition
2. connect as `testadmin/testadmin`
3. load the default metadata templates along with the samples in the GN admin UI
4. visit [Home](http://localhost:8080/home/)
